# Ghostparser

A module for processing user input and generating responses based on predefined pipeline and logic.

## Installation

Clone the repository and install the dependencies:

```bash
pip install -r requirements.txt
```

## Running the Module

To run the module:

```bash
python3 -m ghostparser
```

## Testing

Run tests using pytest:

```bash
pytest
```

## Development

To set up a development environment:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

To run tests:

```bash
pytest -v
```
